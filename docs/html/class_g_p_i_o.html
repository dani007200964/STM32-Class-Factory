Detailed Description

GPIO class layer.

GPIO class is an object layer above HAL_GPIO drivers. With this library you can handle real life things( like a relay, a motor etc... ) as a simple object.

Example code:

// Constructor for a Relay that is connected to
// PA7 on the MCU
GPIO Relay( GPIOA, GPIO_PIN_7 );
int main(){
// Turn on the relay
Relay.on();
// Wait 1000ms
delay( 1000 );
// Turn of the relay
Relay.off();
return 0;
}

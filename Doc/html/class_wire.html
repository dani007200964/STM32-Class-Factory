Detailed Description

Wire I2C Class.

Wire is an I2C library. This is an Arduino Wire like object, the functionality is ment to be the same.

Warning
Slave code is not implemented yet!

uint8_t Wire::endTransmission ( )

Ends a transaction from master.

When you have finished the data sending to the slave you have to end the transaction with this function. In practice this function will send all of the data, the write function just puts it in to a buffer.
